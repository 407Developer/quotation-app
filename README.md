# Quotation Maker (Flooring)

A lightweight, modern web-based quotation tool for generating professional flooring quotations.

This app allows users to calculate accurate costs for flooring installations by inputting room measurements, floor type, skirting options, and door profiles. Each area is calculated independently and added to a running grand total, making it ideal for real-world site measurements.

## Features
- Area-based quotation (multiple rooms)
- Automatic floor area calculation (meters)
- Supports Vinyl and SPC flooring
- Optional skirting, filler, gum, and door profiles
- Real-time grand total calculation
- Clean, modern UI
- Works fully offline once loaded

## Built With
- HTML
- CSS
- Vanilla JavaScript (no framework)

## Use Case
Designed for flooring contractors and interior professionals who need fast, accurate, and presentable quotations on-site or in the office.

## Deployment
Hosted on Netlify for easy access via the web.
